# Serene Libraries

That this project makes use of two Serene Open Source libraries.

They are deployed to this project's codebase, under the src/deployed paths of
the client and server.


## Serene Core

Basic UI components and utility functions.

https://github.com/jfilby/Serene_Core


## Serene AI

Abstraction of OpenAI and Google Gemini AI APIs.

https://github.com/jfilby/Serene_AI
